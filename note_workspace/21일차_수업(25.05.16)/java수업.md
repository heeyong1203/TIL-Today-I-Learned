### interface
- 추상클래스를 extends를 통해 여러 번 적용하는 것이 불가능
- 하지만 현실세계를 반영하는 것과 같이 다기능을 원할 때 필요한 것
- abstract의 추상클래스가 아닌 interface를 활용해보자
- interface는 클래스가 아님
